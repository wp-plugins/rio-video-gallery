== Description ==

A powerful Video Gallery plugin that allows you to embed videos from YouTube, Vimeo and daily motion through categories. You can manage them through a neat and user-friendly settings panel. Also added a help option for user reference.

== Installation ==

This section describes how to install the plugin and get it working in 5 simple steps.

1. Upload `rio-video-gallery` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can see a new section named "Video Gallery" appeared in sidebar.
4. Place the shortcode displayed in settings panel  "[videogallery view="all"]" in any page or post.
5. You are done.

== Frequently Asked Questions ==

= How can i filter video gallery based on a category =

In settings tab under the video gallery, you can generate shortcode based on video categories.
